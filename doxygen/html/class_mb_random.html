Calculates the mean/median values for a discretized gamma.

calculates the average/median values for a discretized gamma distribution

This function calculates the average or median values of the categories for a discretized gamma distribution. The gamma is broken into K categories, with each category having equal probability. The mean or meadian value for each category is then calculated.

Parameters
catRatea pointer to a vector of doubles containing the nCats mean/median values.
athe shape parameter of the gamma distribution.
bthe scale parameter of the gamma distribution.
nCatsthe number of discrete categories.
Whentrue, the median of the category is calculated. Otherwise the mean is used.
Returns
Does not return a value.

◆ lnGamma()

double MbRandom::lnGamma ( double  a)

Natural log of the gamma function.

log of the gamma function

This function calculates the log of the gamma function, which is equal to: Gamma(alp) = {integral from 0 to infinity} t^{alp-1} e^-t dt The result is accurate to 10 decimal places. Stirling's formula is used for the central polynomial part of the procedure.

Parameters
alpis the parameter of the gamma function.
Returns
Returns the log of the gamma function.
Exceptions
Doesnot throw an error.
See also
Pike, M. C. and I. D. Hill. 1966. Algorithm 291: Logarithm of the gamma function. Communications of the Association for Computing Machinery, 9:684.

double MbRandom::uniformRv ( void  )

Uniform[0,1) random variable.

This function generates a uniformly-distributed random variable on the interval [0,1). It is a version of the Marsaglia Multi-Carry.

Taken from: Mathlib : A C Library of Special Functions Copyright (C) 2000, 2003 The R Development Core Team

This random generator has a period of 2^60, which ensures it has the maximum period of 2^32 for unsigned ints (32 bit ints).
