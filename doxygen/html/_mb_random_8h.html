Detailed Description

This file contains a definition for a class for random variables.

In this file we declare a class for random variables. It contains one class, MbRandom, which when instantiated gives a stream of uniform random variables. It also can transform the uniform random variable to generate random variables with different distributions (such as exponential, gamma, normal, etc.).

typedef unsigned int seedType

The Marsaglia Multi-Carry random number generator used in MrBayes 4 assumes a 32-bit data type for seeds. An unsigned int is 32 bits for both 32-bit and 64-bits processors.
