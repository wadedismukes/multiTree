\subsection{Detailed Description}
This file contains a definition for a class for random variables. 

In this file we declare a class for random variables. It contains one class, \mbox{\hyperlink{class_mb_random}{Mb\+Random}}, which when instantiated gives a stream of uniform random variables. It also can transform the uniform random variable to generate random variables with different distributions (such as exponential, gamma, normal, etc.).
